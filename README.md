## 使用方式

### api:

后端程序，使用.Net 6项目框架搭建，运行前请确保安装.net 6 运行时https://dotnet.microsoft.com/en-us/download/dotnet/6.0

```
cd api
```
```
dotnet run
```

### web:

前端程序

```
cd web
```
```
yarn install
```
```
yarn serve
```
